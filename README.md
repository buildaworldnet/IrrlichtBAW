IrrlichtBAW
===========

IrrlichtBAW is a fork of the free 3D engine Irrlicht, maintained by Build A World. IrrlichtBAW currently supports OpenGL 3.3 and 4.0 and is under the [Apache License 2.0](/LICENSE).
